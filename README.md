## sudoku文件夹
放的是生成数独终局和解数独的相关文件和代码，使用的 C 语言
可执行文件 sudoku.exe 放在 sudoku/bin 里面
sudoku/bin 里面 wert.txt 存放数独题目，sudoku.txt 存放生成的数独终局，solve_sudo.txt 存放数独题目的解
wert.txt文件中存放了有1000道数独题目
性能分析报告太大了，放了压缩包
